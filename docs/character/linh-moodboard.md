# Linh — Mood Board Reference Kit

> **Purpose**: starting kit for the PureRef board the artist will use to
> commission Linh. References below are search prompts and curated URL
> patterns, not the assets themselves — copy them into your PureRef
> canvas and prune to taste.
> **Status**: research · cross-reference with `docs/CHARACTER-PLAYBOOK.md`
> §1 (Concept Direction) which defines the 1:3 chibi proportion + 5-color
> palette this board has to embody.

---

## 1. Final visual target — pinpoint references

We want **one believable hire-able character** that reads as
"competent secretary in chibi form, professionally warm". 8–12
references on the final board, ruthlessly curated. Quantity here is
the kit; the artist drops anything that drifts.

Aim for these *exact* qualities, in priority order:

1. 1:3 chibi (SD) head-to-body proportion — head ≈ 1/3 of total height
2. Soft 2-tone cel shading (no airbrush, no realism)
3. Vector-style 2–3 px tapered outlines
4. Adult identity preserved (eye shape ≠ giant moe eyes)
5. Office-attire-but-warm (white blouse + navy skirt, no fan-service)

### 2.2 Anime expression vocabulary

| Source | Query / link pattern |
|---|---|
| Pinterest | `anime soft smile crescent eyes` |
| Pinterest | `anime worry sweat-drop chibi` |
| Pinterest | `anime focused glasses glint` |
| Anime stills | *Working!! / Wagnaria!*, *New Game!*, *Servant × Service*, *Psycho-Pass* (chibi-fied Akane) |

For each Shikigami state, grab 1 reference matching the brief in
`CHARACTER-PLAYBOOK.md` §1.1:

- **idle** — neutral polite, hands clasped at waist (Wagnaria standard wait pose)
- **happy** — smile bloom, blush, hands toward chest (Kobashigawa, *New Game!*)
- **happy_relieved** — eyes-closed exhale, glasses adjustment (post-meeting moment, *Servant × Service*)
- **focused** — sharp eyes scanning, glasses glint (Akane, *Psycho-Pass*)
- **warning** — palm-out wait gesture, sweat-drop (Mia Fey, *Ace Attorney*)
- **sleepy** — half-moon eyes, head nod (Konata-style, *Lucky Star*)

## 3. Anti-references (what to **not** show the artist)

Add a "BANNED" zone on the PureRef canvas. Drop these in to make the
direction unambiguous:

- VRoid auto-exports — too stiff, too 3D
- Generic vector "secretary" stock illustrations — too cold
- Oversized moe eyes that erase the adult age
- Sexualised office tropes (low-cut blouse, micro-skirt) — Linh is warm/competent, not fan-service
- Photo-real / semi-real illustration — wrong stylistic register
- 1:2 "bean" chibi — reads as mascot, not hire-able adult

---

## 4. Output checklist

Before handing the PureRef file to the artist, verify:

- [ ] 8–12 references on canvas, all in the same overall stylistic
      neighbourhood
- [ ] Every Shikigami state has ≥1 expression reference labelled with
      the state name
- [ ] Banned zone has 4–6 anti-references with a red X overlay
- [ ] The hex palette swatches are pasted as a colour bar at top-right
- [ ] One sentence pinned to the canvas summarising the persona — copy
      from `CHARACTER-PLAYBOOK.md` §1: *"competent secretary in chibi
      format, professional but approachable. SFW. Warm, not corporate-
      cold."*
