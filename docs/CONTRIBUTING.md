## Prerequisites

| Tool       | Version  | Why                                |
|------------|----------|------------------------------------|
| Node.js    | ≥ 20     | Vite + React + TypeScript build    |
| pnpm       | ≥ 9      | Package manager (committed lockfile) |
| Rust       | ≥ 1.78   | Tauri 2 backend                    |
| Xcode CLT  | latest   | macOS native build deps            |
| ImageMagick| ≥ 7      | Optional — `reskin-character.sh`   |
| Python 3   | ≥ 3.10   | Hook bridge + install scripts      |

```bash
brew install pnpm rust imagemagick
xcode-select --install
```

## First-Time Setup

```bash
git clone https://github.com/hoangperry/shikigami.git
cd shikigami
pnpm install
./scripts/fetch-hiyori-sample.sh        # default Live2D character
pnpm tauri:dev                           # builds Rust + opens window
```

The first `tauri:dev` compiles the entire dependency graph and may take
several minutes; subsequent launches are incremental (~10s).

## Development Workflow

1. **Plan** — for non-trivial changes, drop a brief in `plans/<date>-<slug>.md`
2. **Branch** — `git checkout -b feat/<short-name>` off `main`
3. **Code** — small commits, conventional format (`feat:`, `fix:`, `docs:`)
4. **Verify locally**:
   ```bash
   pnpm typecheck      # frontend
   cd src-tauri && cargo test --no-default-features --lib
   pnpm lint           # eslint
   ```
5. **Test** — visually via `pnpm tauri:dev` and exercise the change path
6. **PR** — describe intent + test plan; link related ADRs if architectural

### Code style

- TypeScript: kebab-case files, `camelCase` symbols, `PascalCase` components & types
- Rust: snake_case files + symbols, `PascalCase` types — enforced by `cargo fmt`
- Comments explain *why*, not *what* — file headers describe the module's role
- Files ≤ 500 LOC; split into focused modules past that

### Pre-commit checklist

- [ ] `pnpm typecheck` passes
- [ ] `cargo build --no-default-features` (in `src-tauri/`) passes
- [ ] No `console.log` left from debugging
- [ ] No new dependencies without a justifying note in commit body
- [ ] No secrets / API keys in code

## Adding Tests

### Rust (`src-tauri/`)

```rust
#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn event_maps_to_happy() {
        // Arrange
        let evt = ...;
        // Act
        let resolved = resolve(&evt);
        // Assert
        assert_eq!(resolved.dominant, DominantState::Happy);
    }
}
```

Run with `cargo test --no-default-features --lib`.

### TypeScript

Vitest is wired but no suite yet — add `<file>.test.ts` next to source
when introducing a unit-testable utility.

## Adding a New Character

See [CONFIG.md → Character Packages](./CONFIG.md#character-packages) for
the manifest schema. The fastest path:

```bash
./scripts/fetch-live2d-sample.sh Haru                 # add an official sample
./scripts/reskin-character.sh hiyori hiyori-pink pink # recolor an existing one
```

## Reporting Issues

When opening an issue please include:

- Output of `pnpm tauri:dev` (front + back) up to and including the error
- `~/.shikigami/config.json` (redact `tts.api_key` if present)
- macOS version + chip family (Apple Silicon vs Intel)
- Whether reproducible after `rm -rf ~/Library/Application\ Support/dev.shikigami.app`
